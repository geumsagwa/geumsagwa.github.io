피그마(Figma)에서 비네팅(Vignetting) 효과는 사진의 외곽 부분을 어둡게 하여 중앙을 강조하는 기법으로, 주로 Inner Shadow나 Gradient 기능을 활용해 간단히 구현할 수 있습니다. [1, 2] 
1. Inner Shadow를 이용한 방법 (가장 권장)
가장 빠르고 수정이 용이한 방법입니다.

   1. 비네팅을 적용할 프레임이나 이미지를 선택합니다.
   2. 우측 패널의 Effects 섹션에서 +를 눌러 Inner Shadow를 추가합니다.
   3. 효과 설정 아이콘(해 모양)을 클릭하여 값을 조정합니다:
   * X, Y: 0 (중앙 정렬)
      * Blur: 100~200 이상 (부드러운 퍼짐 정도에 따라 조절)
      * Spread: 50 내외 (어두운 영역의 넓이 조절)
      * Color: 검정색(#000000) 또는 어두운 톤, 투명도(Opacity)는 20~50% 정도로 설정합니다. [3, 4] 
   
2. Gradient(그라데이션)를 이용한 방법
특정 방향이나 타원형으로 정교하게 조절하고 싶을 때 유용합니다. [5] 

   1. 이미지 위에 같은 크기의 사각형(Rectangle)을 만듭니다.
   2. Fill 설정에서 색상 타입을 Radial Gradient로 변경합니다.
   3. 그라데이션의 중심점 투명도를 0%로, 외곽점 투명도를 50~80%(검정색)로 설정합니다.
   4. 캔버스에 나타난 그라데이션 핸들을 조절하여 비네팅의 범위를 디자인에 맞게 맞춥니다. [6] 

3. 플러그인 활용
더 복잡한 필터 효과가 필요하다면 Figma Community에서 제공하는 플러그인을 사용할 수 있습니다. [7] 

* [Filter / effects](https://www.figma.com/ko-kr/community/plugin/792025380269016893/filter-effects): 밝기, 대비와 함께 비네팅 강도를 슬라이더로 조절할 수 있습니다. [8] 

추가로 이미지의 특정 부분만 흐리게 하고 싶다면 Progressive Blur 플러그인을 함께 활용해 보세요. [9] 
비네팅의 농도를 조절하고 싶으신가요, 아니면 특정 형태(타원형 등)로 변형하는 방법이 궁금하신가요?

모두 표시
둘 다
비네팅의 농도(강도) 조절과 특정 형태(타원형 등) 변형은 피그마의 레이어 구조와 속성값을 활용하면 매우 정교하게 제어할 수 있습니다.
1. 비네팅 농도(강도) 조절하기
이미 적용된 비네팅이 너무 진하거나 연할 때 다음 두 가지를 확인하세요.
Opacity(투명도) 수치 조정: Inner Shadow의 색상 설정 창에서 알파(A) 값을 조절하거나, 비네팅 전용으로 만든 레이어 자체의 불투명도(0~100%)를 조절하는 것이 가장 직관적입니다.
Blur & Spread 값의 관계:
Blur: 값이 높을수록 경계가 부드러워지며 농도가 연하게 느껴집니다.
Spread: 값이 높을수록 어두운 영역이 안쪽으로 더 많이 침범하며 농도가 진하게 느껴집니다.
2. 특정 형태(타원형/자유형)로 변형하기
단순한 사각형 외곽이 아닌 디자인에 맞춘 타원형 비네팅은 Radial Gradient가 가장 효과적입니다.
이미지 위에 사각형을 덮고 Radial Gradient를 적용합니다.
중앙은 투명(0%), 외곽은 어두운 색(50~80%)으로 설정합니다.
핸들 조작 (핵심): 캔버스에 나타난 그라데이션 가이드 라인을 보세요.
중앙의 원형 포인트를 잡고 늘리면 타원형으로 모양이 변합니다.
끝점의 위치를 옮기면 비네팅이 시작되는 시점과 범위를 자유롭게 조절할 수 있습니다.
자유로운 형태: 만약 비정형적인 모양이 필요하다면, Pen Tool로 모양을 그린 후 Layer Blur 효과를 아주 높게(100 이상) 주어 이미지 위에 얹는 방식을 사용하세요.
지금 제작 중인 디자인이 모바일 앱 화면인가요, 아니면 웹 랜딩 페이지의 메인 비주얼인가요? 용도에 맞춰 더 적합한 수치 가이드를 드릴 수 있습니다.


웹 랜딩 페이지의 메인 비주얼(Hero Section)은 방문자의 시선을 즉시 사로잡아야 하므로, 일반적인 이미지보다 더 부드럽고 웅장한 느낌의 비네팅 처리가 필요합니다. Figma Community의 고품질 템플릿 스타일을 반영한 가이드는 다음과 같습니다.
1. 농도(강도) 조절 가이드
웹 메인 비주얼은 배경 이미지 위에 텍스트(Headline)가 올라가는 경우가 많아, 비네팅이 가독성 확보의 도구가 됩니다.
Inner Shadow 활용 시 추천 수치 (1440px 기준):
Blur: 150 ~ 250 (매우 부드러운 전이를 위해 높게 설정)
Spread: 0 ~ 20 (이미지의 중심부를 해치지 않는 선에서 조정)
Color Opacity: 20% ~ 40% (검정색 대신 브랜드 컬러의 아주 어두운 톤을 쓰면 더 고급스럽습니다)
팁: 비네팅이 부족하다면 효과를 중첩(+ 버튼 클릭)하여 하나는 넓고 연하게, 하나는 좁고 진하게 설정해 보세요.
2. 타원형 및 자유형 변형 가이드
웹 페이지의 해상도는 가로로 길기 때문에(주로 1440px 이상), 정원보다는 가로로 긴 타원형 비네팅이 필수적입니다.
Radial Gradient 조절법:
이미지 위에 사각형을 만든 후 Fill을 Radial로 변경합니다.
형태 변형: 캔버스 중앙의 그라데이션 핸들을 가로로 길게 잡아당겨 타원형으로 만듭니다.
위치 이동: 메인 카피(글자)가 왼쪽에 있다면, 그라데이션의 중심을 약간 오른쪽으로 옮겨 글자 배경을 더 어둡게 눌러줄 수 있습니다.
Layer Blur 활용법:
원을 하나 그린 뒤 Layer Blur 수치를 200 이상으로 높여 이미지 모서리에 배치하세요.
이 방식은 사방을 다 어둡게 하는 대신, 특정 모서리만 어둡게 하여 시선을 유도할 때 유용합니다.
3. 실전 디자인 팁
텍스트 가독성: 비네팅만으로 부족하다면 이미지 위에 검정색에서 투명으로 빠지는 Linear Gradient 레이어를 하단이나 좌측에 추가로 겹치면 가독성이 극대화됩니다.
색상 팁: 완전한 검정(#000000)보다는 이미지의 가장 어두운 부분 색상을 추출해 사용하면 배경과 훨씬 자연스럽게 어우러집니다.
현재 배경으로 쓰시는 이미지가 밝은 톤인가요, 아니면 어두운 밤 풍경 같은 톤인가요? 이미지 톤에 따라 적절한 컬러 블렌딩 모드(Multiply 등)를 추천해 드릴 수 있습니다.
